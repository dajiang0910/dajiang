## Phase 2 / W4-6:结构化输出 + RAG(项目正式启动 → 「企业知识库智能助手」)

### Week 4 — 结构化输出 + 文档解析
**产出物**:上传文档 → 返回结构化要点 JSON(知识库"接入层"雏形)。
**关键点**:`BeanOutputConverter` · Prompt 工程 · JSON Schema · Tika/PDFBox

### Week 5 — RAG v1(知识库问答跑通)
**产出物**:问答能给出答案 + 命中来源片段。
**关键点**:Embedding 直觉 · 向量库起步(Redis-Stack/PGVector)· 文档切分 · `QuestionAnswerAdvisor`

### Week 6 — RAG v2(检索质量工程化)
**产出物**:v2 相对 v1 召回/准确率有可量化提升。
**关键点**:混合检索(BM25+向量)· rerank · Query 改写 · 评测集 · Redis 缓存 · **Milvus**

## Phase 3 / W7-9:Agent + 工作流 + MCP(项目升级为"助手")

**关键点**:工作流编排 · 状态机 · 失败重试 · human-in-the-loop · **Spring AI MCP**

### Week 9 — 主项目闭环打通 ⭐ 第一个里程碑
